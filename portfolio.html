Week 4: Hosting Data

This week we had to scrape data from a website using python. I choose to scrape data from the Birmingham 2022 commonwealth games website, specifically the medal table. After retrieving the table in python I cleaned the data and transformed it into a TIDY format which can be seen in this Python Colab book. I choose data from the commonwealth site because when I inspected the element on the website I could see that the data was stored in the form of a table and would be easy to scrape.
